## 📂 Repository Structure

This repository contains all the R scripts used to generate the main and supplementary figures for the manuscript:

* `Fig1.R` - `fig6.R`: Scripts for generating Main Figures 1 through 6.
* `FigS1.R` - `figS5.R`: Scripts for generating Supplementary Figures.
* `my_function.R`: A collection of custom R functions utilized throughout the analysis pipeline.
